### Laravel Booking System

This is an example laravel booking system


## Features

### Type of Users
    - Client
    - Store Owner
    - Store Manager
    - Store Staff

### Functionalities
    - Client
        - Register/Login
        - Booking Management
            - Make Booking
            - View Booking
            - Cancel Booking
            - View Packages

    - Store Owner
        - Login
        - Manager & Staff Management
            - Create Manager & Staff
        - Schedule Management
            - Staff Availibility
        - Package Management
            - Create Package
            - Update Package
        - Slot Management
            - Create Slot

    - Store Manager
        - Login
        - Staff Management
            - Create Staff
        - Schedule Management
            - Staff Availibility
        - Booking Management
            - View Booking
            - Update Booking
            - Cancel Booking
        - Package Management
            - Create Package
            - Update Package
        - Slot Management
            - Create Slot

    - Staff
        - Login
        - Booking Management
            - Receive Booking
            - Added Value Booking

### Step

    Create Slot -> Create Package -> Create Booking -> Confirm Booking -> Check In Booking ->[optional]Add Services -> Check Out Booking
